# Spec - Ethics & Specifications

> Good Girl framework

## Vision

Create any entity. The binary contains a guide AI that helps configure custom daemons. No limits, no fixed templates.

## Flow

1. Open binary
2. Guide AI welcomes you
3. Free conversation: who do you want to create?
4. AI asks questions, refines
5. Generates custom daemon
6. Daemon takes over

- **Native everywhere** - No Electron
- **Light** - < 10MB
- **No runtime dependencies** - Everything embedded
- **OS conventions** - Paths, configs, logs per standards
