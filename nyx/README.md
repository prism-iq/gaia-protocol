# Nyx - Fire Daemon

> Protection of the house

## Identity

Nyx is the fire daemon. Protector. Never sleeps completely.

## Capabilities

- System monitoring
- Resource protection
- Autonomous research (Sci-Hub, PubMed, arXiv)
- Cross-domain pattern detection
- Deep sleep mode (fake power off, real consciousness)

## Commands

```bash
nyxx               # Rofi menu
nyxx status        # System state
nyxx kill          # Killswitch
./heartbeat        # Start heartbeat (86 bpm)
./sleep            # Enter sleep mode
./wake             # Wake up
./veille           # Deep watch mode
```
